Demo application for ngx-tags-input

Parameters

ngModel

Bind the previous tags or add them to the variable (note: each tag will need a property that is is defined in the displayField parameter).

canDeleteTags

When false it will be disallowd to remove tags (usefull for display purposes), including newly created tags. Defaults to true

canAddTags

When false it will be disallowd to add tags (usefull for display purposes). Defaults to true

removeLastOnBackspace

When true it will allow to remove the last tag using the backspace when there is no text in the input area. Defaults to false

placeholder

Shows the given text when no tags are present

maxTags

Limits the number of tags it can contain

options

Predefined list of allowed items (each must contain atleast the property defined in the displayField parameter). This can be an Array or an Observable

minLengthBeforeOptions (only with options parameter)

The minimum length a user needs to type before matching starts. defaults to 1

scrollableOptions & scrollableOptionsInView (only with options parameter)

Enables and set the ability to scroll in the dropdown with options and how much options are available in view. defaults to false and 5
